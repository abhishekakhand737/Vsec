## 🧠 Problem Statement  

We are living in the **Attention Economy**, where platforms compete for:

> ⏳ **Your Time**

Platforms like **YouTube, TikTok, and Netflix** maximize engagement — not value.

### ❌ Issues:
- Viral content ≠ Valuable content  
- High consumption, low learning  
- No metric for meaningful content  

---

## 💡 Solution – Attention-X AI  

An AI system that answers:

> 🧠 *“Is this content worth your attention?”*

### Output:
- ⚡ Viral Distraction  
- 🧠 Valuable Insight  

---

## 🔍 Features  

- ⚡ Viral Signal Detection  
- 🧠 Wisdom Signal Detection  
- 📊 Attention + Wisdom Score  
- 🎯 Final Classification  
- 🎨 Clean UI

## 🧪 How It Works  

1. Input content
