## 🧠 About The Project

> In today’s digital world, **attention is currency**.

Platforms like YouTube, TikTok, and Netflix are designed to:
- Hook users  
- Maximize watch time  
- Drive endless scrolling  

⚠️ But the problem:
- Viral content ≠ Valuable content  
- Users lose time without gaining knowledge  

---

## 💡 Solution – Attention-X AI

✨ **Attention-X AI** analyzes content and tells you:

🔴 Is it **just viral distraction**  
🟢 Or **actually meaningful & valuable**

---

## 🔍 Core Features

🚀 **Viral Signal Detection**
- Emotional hooks  
- Clickbait patterns  
- Engagement traps  

🧠 **Wisdom Signal Detection**
- Information depth  
- Learning value  
- Cognitive impact  

📊 **Smart Output Dashboard**
- Attention Score  
- Wisdom Score  
- Final Classification
